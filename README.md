# 🎓 AI-Driven Emotion Detection & Learning Support System

## 📌 Overview
This project is an AI-powered Learning Support Platform that detects a student's learning-related emotions from text input and provides personalized, empathetic guidance using NLP, machine learning concepts, and Google Gemini AI.

It helps learners understand their emotional state while studying and offers tailored learning suggestions to improve understanding and motivation.

---

## 🚀 Features

- 🎯 Emotion Detection from student text input  
- 🧠 Supports emotions: Confused, Frustrated, Confident, Curious, Bored  
- 🤖 AI-generated personalized learning responses using Gemini API  
- 📊 Interactive analytics dashboard (Plotly charts)  
- 📝 Session history tracking  
- 🎨 Clean Streamlit web interface  
- 🔁 Fallback system (AI + template responses)  
- 📈 Confidence score visualization  

---

## 🏗️ Tech Stack

- Python  
- Streamlit  
- NLP (Rule-based + ML-ready design)  
- Google Gemini API  
- Pandas, NumPy  
- Plotly (Data Visualization)
